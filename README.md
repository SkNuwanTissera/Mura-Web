# Mura Web

A React + Vite frontend for the Mura backend, built with Material UI.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run dev
   ```
3. Configure backend API URL in `VITE_API_BASE_URL` if needed.

## Notes

- Login is a simple local parent session.
- Activities and children are loaded from the backend.
- Google SSO is intentionally deferred for later.
